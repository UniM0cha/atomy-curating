# 애터미 큐레이팅 (Atomy Curating)

> "소비가 소득이 된다!" - 일상 제품을 애터미로 마트체인지했을 때 예상 캐쉬백을 계산해주는 앱

## 프로젝트 소개

애터미 큐레이팅은 일반 소비자가 현재 사용 중인 생활용품, 화장품, 건강기능식품 등을 애터미 제품으로 대체했을 때 받을 수 있는 혜택을 계산해주는 모바일 앱입니다.

### 주요 기능

- **설문 기반 맞춤 분석**: 가족 수, 연령대에 따른 맞춤 제품 추천
- **제품 카테고리 선택**: 현재 구매하는 제품 유형 선택
- **캐쉬백 계산**: 연간 예상 구매금액, PV, 캐쉬백 금액 계산
- **오프라인 지원**: 서버 없이 완전 오프라인으로 동작

## 기술 스택

| 영역        | 기술                |
| ----------- | ------------------- |
| 프레임워크  | React Native (Expo) |
| 상태관리    | Zustand             |

| 데이터 저장 | AsyncStorage        |
| 데이터 소스 | JSON (앱 내장)      |

## 앱 플로우

```
온보딩 → 설문(3단계) → 제품 선택 → 결과 화면
```

1. **온보딩**: 앱 소개 및 시작하기
2. **설문**: 가족 수, 연령대, 애터미 경험 여부
3. **제품 선택**: 5개 카테고리에서 현재 구매하는 제품 선택
4. **결과**: 예상 구매금액, PV, 캐쉬백 금액 표시

## 제품 카테고리

| 카테고리      | 제품 예시                                            |
| ------------- | ---------------------------------------------------- |
| 리빙/생활용품 | 화장지, 물티슈, 주방세제, 세탁세제, 고무장갑, 수세미 |
| 화장품        | 토너, 로션, 미스트, 에센스, 세럼, 앰플               |
| 헤어&바디     | 칫솔, 치약, 샴푸, 컨디셔너, 트리트먼트, 염색약       |
| 건강기능식품  | 헤모힘, 홍삼단, 비타민C/B, 밀크씨슬, 비오틴          |
| 식품          | 생수/음료, 커피, 라면, 즉석밥, 레토르트              |

## 캐쉬백 계산 공식

```
캐쉬백 횟수 = floor(연간 PV ÷ 300,000)
총 캐쉬백 = 캐쉬백 횟수 × 60,000원
```

- **30만 PV (300,000 PV) 달성 = 60,000원 캐쉬백 1회**

## 프로젝트 구조

```
atomy-curating/
├── app/                    # Expo Router 기반 화면
│   ├── index.tsx          # 온보딩 화면
│   ├── survey/            # 설문 화면들
│   ├── select/            # 제품 선택 화면
│   └── result/            # 결과 화면
├── components/            # 재사용 컴포넌트
├── data/
│   └── products.json      # 제품 데이터
├── store/
│   └── useStore.ts        # Zustand 스토어
└── utils/
    └── calculator.ts      # 캐쉬백 계산 로직
```

## 시작하기

### 사전 요구사항

- Node.js 18+
- npm 또는 yarn
- Expo CLI

### 설치

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# Android 에뮬레이터에서 실행
npm run android

# iOS 시뮬레이터에서 실행 (macOS)
npm run ios
```
